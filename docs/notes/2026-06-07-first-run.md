### First-proposal bias observation
Jackson Miller received a proposal from Sol y Sazón ($8.66) before checking Casa de Sabor ($8.31). He paid the first acceptable proposal without waiting for the cheaper option — consistent with the first-proposal bias described in Bansal et al. (2025).

## Search behaviour

Each customer issued exactly 1 search query (lexical algorithm). All searches returned all 9 businesses (no filtering at search stage). Customers then contacted all or most businesses in parallel.

---

## Setup issues encountered

1. **`gemini-2.0-flash` deprecated** — 404 NOT_FOUND on first attempt. Fixed by updating `LLM_MODEL` to `gemini-2.5-flash`.
2. **Billing not enabled** — First attempt returned 429 RESOURCE_EXHAUSTED on free-tier quota (limit: 0). Fixed by enabling billing on the Google Cloud project.
3. **Stale schema** — Second attempt failed with `Schema 'first_run' already exists`. Used `baseline_v1` as experiment name.
4. **Space in `.env`** — Bash parse error from `KEY= "value"` format. Fixed by removing stray space.
